
# 甲骨文攤位遊戲 - QR Code 使用說明

## 應用程式已修正的問題：

### 1. QR Code 掃描問題已解決
- 新的應用程式URL已經過測試，確保可以正常訪問
- 生成的QR code使用高錯誤糾正能力，即使有輕微損壞也能正常掃描
- 建議將QR code列印尺寸設為至少3x3公分，確保清晰度

### 2. 圖片顯示方式已優化
- 圖片庫現在只顯示文件名，不會顯示實際圖片內容
- 避免玩家在管理頁面意外看到答案
- 支援批次上傳多張圖片

## 使用方法：

### 管理員操作：
1. 掃描QR code或直接訪問網址
2. 點擊「管理圖片庫/設定」
3. 上傳您的甲骨文字圖片
4. 圖片庫會以文件名列表顯示（不會顯示實際圖片）

### 玩家操作：
1. 掃描QR code進入遊戲
2. 點擊「開始遊戲」
3. 系統隨機顯示一張圖片，並開始5秒倒數
4. 5秒後畫面自動變黑
5. 可選擇重新開始或返回主頁

## 技術特點：
- 支援手機直式與橫式瀏覽
- 無需安裝App，直接在瀏覽器中使用
- 不儲存任何檔案，重新整理即恢復預設狀態
- 預設內建3張甲骨文字圖片作為示範

## QR Code 測試建議：
- 使用多種手機型號測試掃描功能
- 確保列印品質清晰
- 建議在不同光線條件下測試掃描效果
